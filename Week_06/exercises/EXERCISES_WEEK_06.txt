#----------------------------------------------------------------------
# Ollama and LLMs
#----------------------------------------------------------------------

1.) Make sure the machine type of your GitHub Codespaces environment is >= 16GB.

2.) Open a new Terminal, start Ollama and list all available LLMs.

    ollama serve
    ollama list

3.) Import the two LLMs: Llama2:latest (7B, 3.8GB) and Llama3.2:latest (3B, 2GB).
    
    ollama run llama2:latest
    ollama run llama3.2:latest

4.) Test the models in the VS Code Terminal.

5.) Stop the LLMs with:

    /bye
